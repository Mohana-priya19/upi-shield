\# UPI Shield — Real-Time UPI Transaction Anomaly Detection with MLOps Pipeline



\## Problem Statement

UPI transaction volume in India has grown massively, and fraud patterns evolve

faster than static rule-based systems can adapt. UPI Shield is an end-to-end

MLOps pipeline that detects anomalous UPI transactions in real time using a

dual-model ensemble (XGBoost + Isolation Forest), tracks experiments with

MLflow, serves predictions via FastAPI, and monitors data drift against real

NPCI transaction statistics to know when retraining is needed.
